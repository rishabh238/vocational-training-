# C & C++ Learning Repository

A repository documenting my journey of learning **C** and **C++** through hands-on coding, object-oriented programming, data structures, algorithms, and practical projects.

Instead of only solving small programming exercises, this repository focuses on **building real applications** to understand programming concepts and software design.

## 🚀 Projects

This repository contains several projects built while learning C and C++.

### 🎮 Full-Scale Cricket Game

A console-based cricket game featuring:

* Toss system
* Batting & Bowling
* Multiple overs
* Match summary
* Randomized gameplay
* Win/Loss determination
* Scoreboard

### 🏏 Cricket Scoring System

A lightweight cricket score tracker that includes:

* Team information
* Runs, wickets, and overs
* Live score updates
* Match statistics
* Final result generation

### 📋 Report Card Management System

A file-based application for managing student records.
Features include:

* Add student records
* Calculate grades
* Percentage calculation
* Search and update records
* File storage
